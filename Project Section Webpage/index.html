Simple Calculator from eval() method

Hey guys! This is my Calculator! This calculator can do almost all operations. This is working on eval function of JavaScript and buttons are arranged by the help of table tag. You can use it by clicking on the buttons (on Desktop) or by simply taping on it (on mobile phones). The result would be displayed at the top of calculator. It's very awesome check it out !!!
